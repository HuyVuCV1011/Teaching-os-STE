## 🔒 Security Policy

For security and privacy, implementation specs containing:
- Exact PostgreSQL database schemas
- Row-Level Security (RLS) policies
- JWT authentication/cookie details
- API integration webhook signatures and contracts

are excluded from the public git history. If you are an active developer or core maintainer, please refer to the private development wiki or secure document storage for the complete specification pack.
